mw.tools

  • .isEmptyObject(Object)

    checks if Object has any properties

  • .migrateAttributes(from, to, except)

    Copies attributes from one element to another. The first to parameters are required. The except attribute is an array with parameters that you don't want to copy

  • .getAttrs(node)

    Collects all attributes from a node in an Object (ex: {title:'Some title'})

  • .get_filename(String)

    Returns file name from a given url string (ex: example.com/image.jpg - returns 'image.jpg')

  • .fullscreen(node)

    Puts node element in fullscreen mode
